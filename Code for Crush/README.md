# Love Code Website

A romantic website purely made of code for your crush.

## Setup Instructions

1. Open your terminal in this directory.
2. Run the following command to install dependencies (if not already installed):
   ```bash
   npm install
   ```
   *Note: If you encounter permission errors, try closing other terminals or running as Administrator.*

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features
- **SQL of Love**: A typing animation of a romantic SQL query.
- **Infinite Loops**: Code snippets representing eternal love.
- **Interactive Terminal**: Run the "relationship" executable to see the result.
