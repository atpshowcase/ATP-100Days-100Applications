## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## How to Play

1. Enter a number between 1 and 100
2. Click "Guess" or press Enter
3. Follow the hints to find the secret number
4. Try to beat your best score!

## Pro Tips

- Start with 50 to divide the range in half
- Pay attention to "very close" hints (within 5 numbers)
- Use binary search strategy for optimal results
- The perfect score is 7 attempts or less!
