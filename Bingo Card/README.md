# Bingo Card App

A premium, glassmorphism-styled Bingo Card application built with Next.js.

## Setup

1. Open this directory in your terminal.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open `http://localhost:3000` in your browser.

## Features

- **Glassmorphism Design**: Modern, translucent UI with vibrant gradients.
- **Interactive Gameplay**: Click to mark cells, auto-detect Win condition.
- **Responsive**: Adapts to different screen sizes.

## Technology Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **CSS Modules** (Vanilla CSS)
- **React 18**
