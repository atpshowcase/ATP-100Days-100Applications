# Storyworthy Journal

A minimalist black and white journaling application inspired by Matthew Dicks' "Storyworthy" method - capturing your life one sentence at a time.

## About

This application helps you build a daily journaling habit by writing just one sentence per day. The Storyworthy method, created by Matthew Dicks, encourages you to reflect on your day and capture the most meaningful moment in a single sentence.

## Getting Started

The development server is already running at [http://localhost:3000](http://localhost:3000)

To start the server manually:

```bash
npm run dev
```

## How to Use

1. **Write Tab**: 
   - Select a date using the navigation buttons
   - Write one sentence that captures your day
   - Click "Save Entry" to store it

2. **History Tab**:
   - View all your past entries in chronological order
   - Click on any entry to edit it

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Storage**: Browser localStorage

## Design Philosophy

The application follows a strict minimalist aesthetic:
- Black and white color scheme only
- Clean typography with system fonts
- Simple borders and spacing
- No unnecessary animations or effects
- Focus on content and writing

## The Storyworthy Method

Each day, write one sentence about the most storyworthy moment. This could be:
- A moment of change
- A realization or insight
- An unexpected event
- A meaningful interaction
- Something that made you feel

Over time, these sentences become a powerful record of your life's story.
