# Age Calculator (Next.js)

A small Next.js app that calculates age in years, months, and days from a birth date input.

## Quick start

1. Install dependencies

   npm install

2. Run the dev server

   npm run dev

3. Open http://localhost:3000 in your browser

## Notes

- Built with Next.js and React (JavaScript)
- Simple front-end only logic (no backend required)
- Feel free to convert to TypeScript or enhance styling
