# Atmosphere Weather App

A beautiful, dynamic weather application built with Next.js and Open-Meteo API.

## Features
- Real-time weather data
- Search by city
- Premium glassmorphism design
- Responsive layout

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack
- Next.js 14
- React
- Lucide React (Icons)
- CSS Modules / Vanilla CSS variables
